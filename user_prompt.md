# GROWMI — Marketing Website Build Prompt

## OVERVIEW

Build a stunning, modern, conversion-optimized marketing website for _Growmi_ — an AI-powered WhatsApp assistant for businesses. The website should look like it belongs to a well-funded startup, NOT a template site. Think Intercom meets Stripe level design quality.

_Tech stack:_ Next.js + Tailwind CSS + Framer Motion (animations). Static site, no backend needed. Deploy on Vercel.

_Brand:_
•⁠ ⁠Name: Growmi
•⁠ ⁠Tagline: "Your AI Employee on WhatsApp"
•⁠ ⁠Domain: getgrowmi.com
•⁠ ⁠Brand colors: Emerald green (#059669) as primary, with dark navy (#0F172A) for headers, white/gray for backgrounds
•⁠ ⁠Logo: Use a WhatsApp-style chat bubble icon with a subtle AI/sparkle element inside. Use the MessageCircle icon from Lucide as placeholder.
•⁠ ⁠Tone: Confident, modern, slightly playful but professional. Speak to business owners, not developers.

## DESIGN SPECIFICATIONS

_Overall Aesthetic:_
•⁠ ⁠Clean, modern, lots of whitespace
•⁠ ⁠Rounded corners on everything (cards, buttons, images)
•⁠ ⁠Subtle shadows, no harsh borders
•⁠ ⁠Gradient accents (emerald → teal)
•⁠ ⁠Smooth scroll animations (elements fade in as you scroll)
•⁠ ⁠Dark sections alternating with light sections for visual rhythm

_Typography:_
•⁠ ⁠Headings: Inter or Geist, bold, large (48px hero, 36px sections, 24px sub-sections)
•⁠ ⁠Body: Inter or Geist, regular, 16-18px, gray-600 color
•⁠ ⁠Use sentence case for headings, not ALL CAPS

_Buttons:_
•⁠ ⁠Primary: Emerald-600, white text, rounded-xl, padding 16px 32px, hover effect (slight scale + darken)
•⁠ ⁠Secondary: White/transparent, emerald border, emerald text, same hover
•⁠ ⁠All buttons should have subtle hover animations

_Cards:_
•⁠ ⁠White background, 1px gray-200 border, rounded-2xl
•⁠ ⁠Subtle shadow on hover
•⁠ ⁠Padding 24-32px

_Animations (Framer Motion):_
•⁠ ⁠Hero text: Slide up + fade in on load (staggered for headline → subheadline → CTA)
•⁠ ⁠Section headings: Fade in when scrolled into view
•⁠ ⁠Cards: Slide up + fade in when scrolled into view (staggered for grid items)
•⁠ ⁠Stats counter: Count up animation when scrolled into view
•⁠ ⁠Phone mockup: Subtle floating animation (translateY oscillation)
•⁠ ⁠Smooth page transitions

_Phone Mockups:_
•⁠ ⁠Use CSS-styled phone frames (rounded rectangle with notch)
•⁠ ⁠Show WhatsApp-style chat UI inside with green/white message bubbles
•⁠ ⁠Messages should have realistic timestamps and blue ticks
•⁠ ⁠Make it look REAL, not like a wireframe

_Responsive Design:_
•⁠ ⁠Must look perfect on mobile (60%+ of Indian traffic is mobile)
•⁠ ⁠Hamburger menu on mobile
•⁠ ⁠Stack columns on mobile
•⁠ ⁠Full-width cards on mobile
•⁠ ⁠Touch-friendly CTAs (minimum 48px tap targets)

_Illustrations/Icons:_
•⁠ ⁠Use Lucide React icons throughout
•⁠ ⁠For industry icons: use relevant Lucide icons (Stethoscope for clinics, Scissors for salons, Dumbbell for gyms, Dog for pet clinics, etc.)
•⁠ ⁠No stock photos — use icons, illustrations, and product screenshots only

_WhatsApp Chat Mockup Component:_
•⁠ ⁠Create a reusable component that renders a fake WhatsApp conversation
•⁠ ⁠Green bubbles (right) for customer messages
•⁠ ⁠White bubbles (left) for AI responses
•⁠ ⁠Show timestamp, blue ticks, typing indicator
•⁠ ⁠Support for emoji rendering
•⁠ ⁠Different conversations for different use cases
